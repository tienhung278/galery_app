# Frontend
## Galery app
### Step 1
Install node.js
### Step 2
Open command line
### Step 3
Run command `npm install -g @angular/cli`
### Step 4
Change to root folder
### Step 5
Run command `npm install`
### Step 6
Run command `npm start`
### Step 7
Open the browser
### Step 8
Access the url `https://localhost:4200`
